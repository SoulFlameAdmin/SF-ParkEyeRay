# SmartCity V2 production release

Promoted to `main` on 2026-07-16.

Release includes:
- mandatory onboarding before the map;
- test Google/guest profile flow;
- honest CarTag/Web NFC capability check and demo mode;
- SmartCity loading transition;
- Waze-style map-first interface and compact burger menu;
- automatic nearby parking and fuel station layers;
- destination parking, driving and walking routes;
- PostGIS parking data engine;
- parking proposal and moderation contracts.
